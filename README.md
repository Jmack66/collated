# Collated

A curated collection of sources, organised by tag.

---

## Tag Reference

Quick-scan guide for tagging new sources. Tags split into **core** (broad, used often) and **niche** (specific, used selectively).

### Core Tags

| Tag | When to use |
|-----|-------------|
| `art` | Visual art, illustration, sculpture, general creative work |
| `design` | Design thinking, product / industrial / graphic design |
| `engineering` | Applied engineering concepts, analysis, builds |
| `fabrication` | Making physical things — sheet, weld, join, assembly |
| `machining` | Mills, lathes, turning, precision metal cutting |
| `electronics` | Circuits, PCBs, components, embedded systems |
| `software` | Programs, dev tools, coding techniques |
| `tools` | Commercial tools — reviews, usage, setup |
| `tooling` | Jigs, fixtures, custom tools for making |
| `workshop` | Shop layout, storage, benches, infrastructure |
| `workspace` | Desks, computer setups, office ergonomics |
| `maker` | General DIY/maker content spanning multiple disciplines |
| `guide` | How-tos, tutorials, step-by-step references |
| `repair` | Fixing things, diagnostics, maintenance |
| `research` | Deep dives, papers, method-heavy content |
| `productivity` | Tools and ideas for doing more / better work |
| `reading` | Articles, books, longform worth revisiting |
| `youtube` | Channels, series, or creators worth tracking |
| `filmmaking` | Camera, lighting, directing, editing craft |
| `vehicles` | Anything vehicle-scale: cars, trucks, bikes, etc. |
